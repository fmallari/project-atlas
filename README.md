# Project Atlas

## Mission

Project Atlas documents my transition into Cloud Engineering through hands-on AWS infrastructure projects.

Rather than completing isolated tutorials, every lesson is treated as a production engineering ticket.

Each ticket includes:

- Business requirement
- Architecture
- Implementation
- Validation
- Incident investigation
- Executive summary
- Lessons learned
- Operational runbook
